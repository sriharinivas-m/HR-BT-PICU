Extension:
==========
This study explores the non-linear relationship between heart rate, body temperature, and age in critically ill children using Gradient Boosting with Quantile Regression, achieving a quantile loss of 6.6 and demonstrating superior predictive accuracy over linear models for improved clinical decision-making in PICU settings.

As an extension, a XGBoost-based model was implemented, further improving performance with a reduced quantile loss of 5.8, enhancing predictive accuracy for clinical decision-making in PICU settings.
Additionally, we employed XAI techniques such as LIME and SHAP to explain the importance of feature values used in the predictions.

Along with this extension we used Flask Framework for UI for user provides input for prediction using trained models for dataset.


Note:
------
In the base paper, the authors compared model performance using quantile loss as the primary evaluation metric. Therefore, we also evaluated our model based on quantile loss rather than other regression metrics. Although we attempted to improve additional performance measures, the dataset used in the base paper is not publicly available, making it impossible to reproduce their results exactly. Nevertheless, when compared with other baseline models, our extended model demonstrates superior predictive performance.
